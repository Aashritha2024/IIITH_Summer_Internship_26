
Task 3 – YOLO Object Detection Model Training

This project involves training a custom YOLOv8 object detection model using a labeled dataset. The goal is to detect objects (e.g., cars, buses) from images/videos with high accuracy using deep learning.

The model was trained using the Ultralytics YOLO framework and evaluated using standard metrics such as Precision, Recall, and mAP.

Classes Used:
car 
bus 

Model Details:
Model: YOLOv8n
Framework: Ultralytics YOLO
Input Image Size: 384x384
Epochs: 100

Training Results:
Overall Performance:
Precision (P): 0.953
Recall (R): 0.966
mAP@0.5: 0.99
mAP@0.5:0.95: 0.742
